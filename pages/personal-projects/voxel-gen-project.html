Voxel Terrain Gen

Information

This is a project I created back in 2022 to try my hand at making a procedural voxel terrain system. The project was created using the Godot game engine (3.5). This project was practice for learning how to generate voxel style terrains. The code was done in C# and is multi-threaded to prevent stutter/freezing while chunks of the terrain are generated. The entire terrain also generates with collisions alongside the mesh of the terrain.
