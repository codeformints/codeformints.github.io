FPS/Puzzle Platformer Game

Information

This is a project I created in 3 days for a game jam. It was the first time I ever participated in a game jam, so it was a bit of a new challenge trying to create an entire game in 3 days. The game uses a tool that can manipulate rigid bodies and propel the player off of certain platforms. I wanted to see just how much I could make on my own within that time frame and created the 3D assets and SFX, along with using a tool called Material Maker to procedurally create materials. All of the code was written in GDScript since I needed a web build and C# was not supported for web builds in Godot 4.3 - 4.4.
